Power Rule

Given the function $y=ax^n$, the derivative is:

$$\frac{dy}{dx}=nax^{n-1}$$

Use this rule to differentiate:

  1. $y=5x^4$
  2. $y=7x^2$
  3. $y=\frac{2}{3}x^6$

These indices facts will be useful for the next questions:

$x^{-n} = \frac{1}{x^n}$
$x^{\frac{1}{n}} = \sqrt[n]{x}$
$x^a \times x^b = x^{a+b}$
$\frac{x^a}{x^b} = x^{a-b}$
$(x^a)^b = x^{ab}$

Use the power rule to differentiate:

  1. $y=\frac{3x^3}{x^5}$
  2. $y=\frac{10}{x^2}$
  3. $y=2x^{\frac{2}{3}}$
  4. $y=4\sqrt{x}$
  5. $y=\frac{2}{3x^2}$
  6. $y=\frac{10}{x}$

The power rule can be used to show:

$y=ax, \quad \frac{dy}{dx}=a$
$y=a, \quad \frac{dy}{dx}=0$

Put this altogether and differentiate:

  1. $y=5x^2+4x+3$
  2. $y=x^4-2x^{-5}+\frac{1}{x}$
  3. $y=1+2\sqrt{x}+\frac{2}{5x}+\frac{3}{4x^2}+\frac{5}{2x^3}$

Logarithms

Given the function $y=a\ln x$, the derivative is:

$$\frac{dy}{dx}=\frac{a}{x}$$

Differentiate:

  1. $y=5\ln x$
  2. $y=\frac{\ln x}{2}-3e^x$

The following log laws will be useful:

$\ln(AB)=\ln A+\ln B$
$\ln\left(\frac{A}{B}\right)=\ln A-\ln B$
$\ln(A^n)=n\ln A$

Differentiate:

  1. $y=\ln 4x$
  2. $y=\ln 0.443x$
  3. $y=\ln \frac{2}{x}$
  4. $y=\ln x^{10}$
  5. $y=6\ln \frac{1}{2}x$

Consumer Optimisation Extension using Partial Differentiation

A consumer aims to maximize their utility from consuming two goods, $x$ and $y$. Their utility function is given by:

$$U(x,y) = x^{0.8}y^{0.2}$$

The prices of the goods are $P_x = 4$ and $P_y = 2$, and the consumer has an income of $I = 200$.

Part of a solution:

Budget Constraint:

$$4x + 2y = 200$$

Lagrangian:

$$L(x,y,\lambda) = x^{0.8}y^{0.2} - \lambda(200 - 4x - 2y)$$

  1. The Lagrangian function is provided above.

First Order Conditions:

Differentiate the Lagrangian with respect to $x$, $y$, and $\lambda$, and set each derivative to zero. Remember to apply the power rule for differentiation.

$$\frac{\partial L}{\partial x} = \text{ (Differentiate L with respect to x, holding y and } \lambda \text{ constant.)} = 0$$ $$\frac{\partial L}{\partial y} = \text{ (Differentiate L with respect to y, holding x and } \lambda \text{ constant.)} = 0$$ $$\frac{\partial L}{\partial \lambda} = \text{ (Differentiate L with respect to } \lambda \text{, holding x and y constant.)} = 0$$

  1. Write down the full first-order conditions, performing the differentiation steps.

Solving for Optimal Quantities:

  1. You can rearrange $\frac{\partial L}{\partial x}$ to get: $0.8x^{-0.2}y^{0.2} = 4\lambda$. Now, create a similar rearranged expression for $\frac{\partial L}{\partial y}$.

Dividing the Equations:

You then divide these two rearranged equations by one another:

$$\frac{0.8x^{-0.2}y^{0.2}}{\boxed{\phantom{0.2x^{0.8}y^{-0.8}}}} = \frac{4\lambda}{\boxed{\phantom{2}}\lambda}$$

  1. Substitute your rearranged expression for $\frac{\partial L}{\partial y}$ from step 3 into the ratio above.
  2. Simplify the resulting equation to express $y$ in terms of $x$.
    Tip: Gather terms involving $x$ on one side and terms involving $y$ on the other. Remember to simplify numerical coefficients and combine terms with common bases (e.g., $\frac{A^m}{A^n} = A^{m-n}$).
  3. Substitute your equation for $y$ into the budget constraint ($4x + 2y = 200$).
  4. Solve the resulting equation for $x$.
  5. Use the optimal $x$ value to find the optimal quantity of $y$.
