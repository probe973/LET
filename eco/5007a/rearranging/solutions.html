Skills - Rearranging Equations Solutions

  1. Original: $x+y=12$

    Subtract $y$ from both sides:

    $$x = 12-y$$

  2. Original: $5x-4=y$

    Add 4 to both sides:

    $$5x = y+4$$

    Divide by 5:

    $$x = \frac{y+4}{5}$$

  3. Original: $ax-3by=9$

    Add $3by$ to both sides:

    $$ax = 9+3by$$

    Divide by $a$:

    $$x = \frac{9+3by}{a}$$

  4. Original: $y=mx+c$

    Subtract $c$ from both sides:

    $$y-c = mx$$

    Divide by $m$:

    $$x = \frac{y-c}{m}$$

  5. Original: $2+5xy=6$

    Subtract 2 from both sides:

    $$5xy = 4$$

    Divide by $5y$:

    $$x = \frac{4}{5y}$$

  6. Original: $9x+6y=4x+1$

    Subtract $4x$ from both sides and subtract $6y$ from both sides:

    $$9x-4x = 1-6y$$

    Simplify:

    $$5x = 1-6y$$

    Divide by 5:

    $$x = \frac{1-6y}{5}$$

  7. Original: $ax+by=4x+1$

    Subtract $4x$ from both sides and subtract $by$ from both sides:

    $$ax-4x = 1-by$$

    Factor out $x$:

    $$x(a-4) = 1-by$$

    Divide by $(a-4)$:

    $$x = \frac{1-by}{a-4}$$

  8. Original: $ax-by=bx-8$

    Subtract $bx$ from both sides and add $by$ to both sides:

    $$ax-bx = by-8$$

    Factor out $x$:

    $$x(a-b) = by-8$$

    Divide by $(a-b)$:

    $$x = \frac{by-8}{a-b}$$

  9. Original: $\frac{2ty+mx}{ty-mx}=1$

    Multiply both sides by $(ty-mx)$:

    $$2ty+mx = ty-mx$$

    Add $mx$ to both sides and subtract $2ty$ from both sides:

    $$mx+mx = ty-2ty$$

    Simplify:

    $$2mx = -ty$$

    Divide by $2m$:

    $$x = \frac{-ty}{2m}$$

  10. Original: $\frac{ty+mx}{ty-3mx}=k$

    Multiply both sides by $(ty-3mx)$:

    $$ty+mx = k(ty-3mx)$$

    Distribute $k$:

    $$ty+mx = kty-3kmx$$

    Gather terms with $x$ on one side, others on the other:

    $$mx+3kmx = kty-ty$$

    Factor out $x$ on the left and $ty$ on the right:

    $$x(m+3km) = ty(k-1)$$

    Divide by $(m+3km)$:

    $$x = \frac{ty(k-1)}{m(1+3k)}$$

  11. Original: $9x^2=4y$

    Divide by 9:

    $$x^2 = \frac{4y}{9}$$

    Take the square root of both sides (remembering $\pm$):

    $$x = \pm\sqrt{\frac{4y}{9}}$$

    Simplify the square root:

    $$x = \pm\frac{2\sqrt{y}}{3}$$

  12. Original: $x^3-5m=t$

    Add $5m$ to both sides:

    $$x^3 = t+5m$$

    Take the cube root of both sides:

    $$x = \sqrt[3]{t+5m}$$

  13. Original: $x^2-2x=15$

    Rearrange to standard quadratic form $ax^2+bx+c=0$:

    $$x^2-2x-15=0$$

    Factor the quadratic (or use the quadratic formula):

    $$(x-5)(x+3)=0$$

    Set each factor to zero to find solutions for $x$:

    $$x-5=0 \Rightarrow x=5$$

    $$x+3=0 \Rightarrow x=-3$$

    Therefore:

    $$x=5 \text{ or } x=-3$$

  14. Original: $x^2 = 2 - qx$

    Rearrange to standard quadratic form $ax^2+bx+c=0$:

    $$x^2 + qx - 2 = 0$$

    Use the quadratic formula $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$, where $a=1$, $b=q$, $c=-2$:

    $$x = \frac{-q \pm \sqrt{q^2 - 4(1)(-2)}}{2(1)}$$

    Simplify:

    $$x = \frac{-q \pm \sqrt{q^2 + 8}}{2}$$

  15. Original: $x^2-8=kx$

    For a bonus mark on this part: what must be true for this expression to have real solutions?

    Rearrange to standard quadratic form $ax^2+bx+c=0$:

    $$x^2-kx-8=0$$

    For real solutions, the discriminant ($\Delta = b^2 - 4ac$) must be greater than or equal to zero ($\Delta \ge 0$).

    Here, $a=1$, $b=-k$, $c=-8$.

    Substitute these values into the discriminant formula:

    $$\Delta = (-k)^2 - 4(1)(-8)$$

    $$\Delta = k^2 + 32$$

    For real solutions, we need:

    $$k^2 + 32 \ge 0$$

    Since $k^2$ is always non-negative ($k^2 \ge 0$) for any real value of $k$, adding 32 will always result in a positive number ($k^2+32 > 0$).

    Therefore, this expression will always have real solutions for any real value of $k$.

Rearranging with Indices Solutions

  1. Original: $x^{\frac{3}{4}}=y^{\frac{1}{2}}$

    Raise both sides to the power of $\frac{4}{3}$ (the reciprocal of $\frac{3}{4}$):

    $$(x^{\frac{3}{4}})^{\frac{4}{3}}=(y^{\frac{1}{2}})^{\frac{4}{3}}$$

    $$x = y^{\frac{1}{2} \times \frac{4}{3}}$$

    $$x = y^{\frac{4}{6}}$$

    $$x = y^{\frac{2}{3}}$$

  2. Original: $x^{\frac{3}{4}}=y^{\frac{1}{2}}-2$

    Raise both sides to the power of $\frac{4}{3}$:

    $$(x^{\frac{3}{4}})^{\frac{4}{3}}=(y^{\frac{1}{2}}-2)^{\frac{4}{3}}$$

    $$x = (y^{\frac{1}{2}}-2)^{\frac{4}{3}}$$

  3. Original: $5x^8-y=0$

    Add $y$ to both sides:

    $$5x^8 = y$$

    Divide by 5:

    $$x^8 = \frac{y}{5}$$

    Take the 8th root of both sides (remembering $\pm$ for even roots):

    $$x = \pm\left(\frac{y}{5}\right)^{\frac{1}{8}}$$

  4. Original: $x^{\frac{3}{4}}y=8my$

    Divide both sides by $y$ (assuming $y \ne 0$):

    $$x^{\frac{3}{4}} = 8m$$

    Raise both sides to the power of $\frac{4}{3}$:

    $$(x^{\frac{3}{4}})^{\frac{4}{3}} = (8m)^{\frac{4}{3}}$$

    $$x = (8m)^{\frac{4}{3}}$$

  5. Original: $2x^{2.5}=64$

    Divide by 2:

    $$x^{2.5} = 32$$

    Rewrite 2.5 as a fraction $\frac{5}{2}$:

    $$x^{\frac{5}{2}} = 32$$

    Raise both sides to the power of $\frac{2}{5}$:

    $$(x^{\frac{5}{2}})^{\frac{2}{5}} = (32)^{\frac{2}{5}}$$

    $$x = (2^5)^{\frac{2}{5}}$$

    $$x = 2^{5 \times \frac{2}{5}}$$

    $$x = 2^2$$

    $$x = 4$$

  6. Original: $x^2y^3x^5y^2=a$

    Combine terms with the same base using $x^a x^b = x^{a+b}$ and $y^a y^b = y^{a+b}$:

    $$x^{2+5}y^{3+2}=a$$

    $$x^7y^5=a$$

    Divide by $y^5$:

    $$x^7 = \frac{a}{y^5}$$

    Take the 7th root of both sides:

    $$x = \left(\frac{a}{y^5}\right)^{\frac{1}{7}}$$

  7. Original: $0.5x^{-0.5}y^{-0.5} \times 0.5x^{-0.5}y^{0.5}=4$

    Combine numerical coefficients and terms with the same base:

    $$(0.5 \times 0.5) \times x^{-0.5+(-0.5)} \times y^{-0.5+0.5} = 4$$

    $$0.25 \times x^{-1} \times y^0 = 4$$

    Since $y^0=1$:

    $$0.25x^{-1} = 4$$

    Rewrite $x^{-1}$ as $\frac{1}{x}$: $$\frac{0.25}{x} = 4$$

    Multiply by $x$: $$0.25 = 4x$$

    Divide by 4: $$x = \frac{0.25}{4}$$

    $$x = 0.0625$$

  8. Original: $0.8y^{-0.2}x^{0.2} \times 0.2y^{0.8}x^{-0.8}=10$

    Combine numerical coefficients and terms with the same base:

    $$(0.8 \times 0.2) \times y^{-0.2+0.8} \times x^{0.2+(-0.8)} = 10$$

    $$0.16 \times y^{0.6} \times x^{-0.6} = 10$$

    Rewrite $x^{-0.6}$ as $\frac{1}{x^{0.6}}$:

    $$\frac{0.16y^{0.6}}{x^{0.6}} = 10$$

    Multiply by $x^{0.6}$:

    $$0.16y^{0.6} = 10x^{0.6}$$

    Divide by 10:

    $$\frac{0.16y^{0.6}}{10} = x^{0.6}$$

    $$0.016y^{0.6} = x^{0.6}$$

    Raise both sides to the power of $\frac{1}{0.6}$ (or $\frac{10}{6} = \frac{5}{3}$):

    $$x = (0.016y^{0.6})^{\frac{1}{0.6}}$$

    $$x = (0.016)^{\frac{5}{3}} y^{(0.6 \times \frac{5}{3})}$$

    $$x = (0.016)^{\frac{5}{3}} y$$

  9. Original: $\frac{\alpha y^{\alpha-1}x^{\beta}}{\beta y^{\alpha}x^{\beta-1}}=k$

    Separate coefficients and terms with the same base:

    $$\frac{\alpha}{\beta} \times \frac{y^{\alpha-1}}{y^{\alpha}} \times \frac{x^{\beta}}{x^{\beta-1}} = k$$

    Apply index rule $\frac{A^m}{A^n} = A^{m-n}$:

    $$\frac{\alpha}{\beta} \times y^{(\alpha-1)-\alpha} \times x^{\beta-(\beta-1)} = k$$

    $$\frac{\alpha}{\beta} \times y^{-1} \times x^1 = k$$

    Rewrite $y^{-1}$ as $\frac{1}{y}$: $$\frac{\alpha x}{\beta y} = k$$

    Multiply by $\beta y$: $$\alpha x = k \beta y$$

    Divide by $\alpha$: $$x = \frac{k \beta y}{\alpha}$$

Multivariate Optimisation Rearranging Practice Solutions

Zy is a pet dog, who gains utility from eating and from playing catch. Her utility function is defined as:

$$U(E,C)=E^{0.6}C^{0.4}$$

Where $E$ is the number of hours eating and $C$ is the number of hours playing catch.

During the day, Zy has a maximum of 8 hours for eating and catching.

For maximum utility, how many hours should Zy spend eating and catching, to the nearest minute?

Part of a solution:

Constraint function:

$$E+C=8$$

Langrangian:

$$L=E^{0.6}C^{0.4}-\lambda(8-\cdots)$$

  1. Replace the dots with the rearranged constraint function.

    The constraint function is $E+C=8$. To fit it into the Lagrangian form $\lambda(\text{constant} - \text{expression})$, we rearrange it to $8 - E - C$.

    So the Lagrangian becomes:

    $$L=E^{0.6}C^{0.4}-\lambda(8-E-C)$$

First Order Conditions:

$$\frac{dL}{dE}=0.6E^{-0.4}C^{0.4}-\lambda=0$$ $$\frac{dL}{dC}=0.4E^{0.6}C^{-0.6}-\lambda=0$$

  1. For both first order conditions, rearrange to make $\lambda$ the subject of an equation in $E$ and $C$.

    From $\frac{dL}{dE}=0.6E^{-0.4}C^{0.4}-\lambda=0$:

    $$\lambda = 0.6E^{-0.4}C^{0.4}$$

    From $\frac{dL}{dC}=0.4E^{0.6}C^{-0.6}-\lambda=0$:

    $$\lambda = 0.4E^{0.6}C^{-0.6}$$

Dividing the Equations:

You then see that they have divided these two equations by one another:

$$\frac{0.6E^{-0.4}C^{0.4}}{0.4E^{0.6}C^{-0.6}}=\frac{\lambda}{\lambda}$$

  1. Make $C$ the subject of this equation. You will need to use rules of indices.

    Simplify the numerical coefficients and apply index rules ($\frac{A^m}{A^n} = A^{m-n}$):

    $$\frac{0.6}{0.4} \times \frac{E^{-0.4}}{E^{0.6}} \times \frac{C^{0.4}}{C^{-0.6}} = 1$$

    $$1.5 \times E^{-0.4-0.6} \times C^{0.4-(-0.6)} = 1$$

    $$1.5 E^{-1} C^{1} = 1$$

    Rewrite $E^{-1}$ as $\frac{1}{E}$: $$1.5 \frac{C}{E} = 1$$

    Multiply by $E$: $$1.5 C = E$$

    Divide by 1.5 (or multiply by $\frac{1}{1.5} = \frac{2}{3}$): $$C = \frac{E}{1.5}$$

    $$C = \frac{2}{3}E$$

  2. Substitute your equation for $C$ into the constraint function $E+C=8$.

    Substitute $C = \frac{2}{3}E$ into $E+C=8$:

    $$E + \frac{2}{3}E = 8$$

  3. Solve this equation for $E$.

    Combine like terms:

    $$\frac{3}{3}E + \frac{2}{3}E = 8$$

    $$\frac{5}{3}E = 8$$

    Multiply by $\frac{3}{5}$: $$E = 8 \times \frac{3}{5}$$

    $$E = \frac{24}{5}$$

    $$E = 4.8 \text{ hours}$$

  4. Hence find the values of $E$ and $C$ that maximise utility for Zy and convert your answer to hours and minutes (e.g., 3.5 hours is 3 hours and 30 minutes).

    Using $E=4.8$ hours, find $C$ using $C = \frac{2}{3}E$:

    $$C = \frac{2}{3} \times 4.8$$

    $$C = 2 \times 1.6$$

    $$C = 3.2 \text{ hours}$$

    Convert to hours and minutes:

    For $E=4.8$ hours:

    4 hours and $(0.8 \times 60)$ minutes = 4 hours and 48 minutes.

    For $C=3.2$ hours:

    3 hours and $(0.2 \times 60)$ minutes = 3 hours and 12 minutes.

    Therefore, Zy should spend 4 hours and 48 minutes eating and 3 hours and 12 minutes playing catch to maximize utility.

Challenge Problem: Multivariate Optimisation Question Solution

A utility function is given: $U=x^{\frac{1}{2}} y^{\frac{1}{2}}$

With constraint: $5x+4y=20$

Find values for $x$ and $y$ that maximise the utility function.

Step 1: Set up the Lagrangian Function

$$L = x^{\frac{1}{2}} y^{\frac{1}{2}} - \lambda(5x+4y-20)$$

Step 2: Find the First Order Conditions (FOCs)

Differentiate $L$ with respect to $x$, $y$, and $\lambda$ and set them to zero.

$$\frac{\partial L}{\partial x} = \frac{1}{2}x^{-\frac{1}{2}}y^{\frac{1}{2}} - 5\lambda = 0 \quad (1)$$

$$\frac{\partial L}{\partial y} = \frac{1}{2}x^{\frac{1}{2}}y^{-\frac{1}{2}} - 4\lambda = 0 \quad (2)$$

$$\frac{\partial L}{\partial \lambda} = -(5x+4y-20) = 0 \Rightarrow 5x+4y=20 \quad (3)$$

Step 3: Solve for $\lambda$ from (1) and (2)

From (1):

$$5\lambda = \frac{1}{2}x^{-\frac{1}{2}}y^{\frac{1}{2}}$$

$$\lambda = \frac{1}{10}x^{-\frac{1}{2}}y^{\frac{1}{2}} \quad (4)$$

From (2):

$$4\lambda = \frac{1}{2}x^{\frac{1}{2}}y^{-\frac{1}{2}}$$

$$\lambda = \frac{1}{8}x^{\frac{1}{2}}y^{-\frac{1}{2}} \quad (5)$$

Step 4: Equate the expressions for $\lambda$

Set (4) equal to (5):

$$\frac{1}{10}x^{-\frac{1}{2}}y^{\frac{1}{2}} = \frac{1}{8}x^{\frac{1}{2}}y^{-\frac{1}{2}}$$

Multiply both sides by $10 \times 8 = 80$ to clear denominators:

$$8x^{-\frac{1}{2}}y^{\frac{1}{2}} = 10x^{\frac{1}{2}}y^{-\frac{1}{2}}$$

Rearrange to group $x$ and $y$ terms. Divide by $x^{-\frac{1}{2}}$ (which is equivalent to multiplying by $x^{\frac{1}{2}}$) and by $y^{-\frac{1}{2}}$ (which is equivalent to multiplying by $y^{\frac{1}{2}}$):

$$8 \frac{y^{\frac{1}{2}}}{y^{-\frac{1}{2}}} = 10 \frac{x^{\frac{1}{2}}}{x^{-\frac{1}{2}}}$$

$$8y^{1} = 10x^{1}$$

$$8y = 10x$$

Simplify and solve for $y$ in terms of $x$: $$y = \frac{10}{8}x$$

$$y = \frac{5}{4}x \quad (6)$$

Step 5: Substitute (6) into the Constraint Equation (3)

$$5x+4y=20$$

$$5x+4\left(\frac{5}{4}x\right)=20$$

$$5x+5x=20$$

$$10x=20$$

Step 6: Solve for $x$

$$x = \frac{20}{10}$$

$$x = 2$$

Step 7: Solve for $y$ using the value of $x$

Substitute $x=2$ into $y = \frac{5}{4}x$:

$$y = \frac{5}{4}(2)$$

$$y = \frac{10}{4}$$

$$y = 2.5$$

Conclusion:

The values for $x$ and $y$ that maximize the utility function are:

$$x = 2$$

$$y = 2.5$$
